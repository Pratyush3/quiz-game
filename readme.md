# Quiz Game

A terminal-based quiz game built in Python.

## Features
- Multiple categories (F1, Football, General)
- 15 second timer per question
- Wrong answer summary at the end

## How to run
```bash
python main.py
```

## Tech used
- Python 3.13
- time module

## What I learned
- Dictionary structure for storing data
- Modular code organisation
- Time module for countdowns
